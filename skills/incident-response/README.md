# 🔍 Incident Response & Forensics Skills (10)

> Blue team toolkit for forensics, threat hunting, malware analysis, and incident response.

## Overview

10 skills covering the full incident response lifecycle from detection to recovery.

## Categories

| Category | Skills | Focus |
|---|---|---|
| **Forensics** | 3 | Memory, disk, and cloud forensics |
| **Malware Analysis** | 2 | Static + dynamic analysis, YARA rules |
| **Threat Hunting** | 2 | Hypothesis-driven hunting, threat intel |
| **Incident Response** | 3 | Playbooks, event log analysis, network traffic |

## Quick Start

```
"Analyze this memory dump for rootkit indicators"
"Hunt for lateral movement in Windows Event Logs"
"Write a YARA rule to detect this malware family"
"Investigate this AWS CloudTrail log for compromise"
```

## Install Only Incident Response

```bash
./install.sh --category ir
```
